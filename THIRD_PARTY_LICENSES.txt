FFmpeg
--------------------------------------------------------------------------------

Website:  https://ffmpeg.org
License:  LGPL v2.1+ or GPL v2+ (depending on build configuration)
Source:   https://github.com/FFmpeg/FFmpeg

FFmpeg is used by MediaParser.exe to generate thumbnails for video and image
files. MediaParser.exe calls the external ffmpeg.exe CLI tool via CreateProcessW
to extract single frames and resize images. FFmpeg is NOT statically or
dynamically linked into any LlaShell binary.

How FFmpeg is called:
  - File:       MediaParser/src/mediaparser_main.cpp
  - Function:   MediaParserService::TryFFmpegThumbnail()
  - Mechanism:  CreateProcessW("ffmpeg.exe -i <input> -vframes 1 -vf scale=W:H output.bmp")
  - Fallback:   If ffmpeg.exe is not found, MediaParser falls back to
                SHGetFileInfoW shell icon thumbnails.

License compliance notes:
  - LlaShell does NOT modify FFmpeg source code.
  - LlaShell does NOT link against FFmpeg libraries (static or dynamic).
  - FFmpeg is invoked as an independent process, communicating only via
    temporary files on disk.

    FFmpeg's license terms. If your ffmpeg.exe build includes GPL-licensed
    components (e.g., x264, x265), you must comply with GPL v2.
  - If your ffmpeg.exe is a pure LGPL build, the LGPL v2.1 terms apply.
  - Source code for FFmpeg is available at https://github.com/FFmpeg/FFmpeg
